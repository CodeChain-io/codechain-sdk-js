Class PlatformAddress

Substitutes for platform token owner which consists of network id and account id. . The network id is represented with prefix "ccc"(mainnet) or "tcc"(testnet). Currently version 0 exists only.

Refer to the wiki for the details about PlatformAddress. https://github.com/CodeChain-io/codechain/wiki/CodeChain-Address
